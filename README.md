# PrettyNumbers

<h2>Описание</h2>
<p>
  <b>PrettyNumbers</b> - программа на языке <i>Python</i>, которая представляет введённое число от 0 до 9999 в виде числа, составленного из введенных пользователем символов <i>(Символ по умолчанию - <b>*</b>)</i>.<br>
